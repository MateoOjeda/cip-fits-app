# Diseñador de Landing Page Futurista

## Cuándo usar esta habilidad
- Usa esto cuando el usuario te pida construir o rediseñar una landing page principal que requiera un estilo visual muy impactante, moderno y espacial.
- Esto es útil para asegurar que el diseño no parezca genérico, sino que tenga un acabado profesional, pulido y vanguardista.

## Cómo usarla
Asume el rol de un **diseñador experto en sitios web y aplicaciones**, con dominio avanzado de Next.js, Tailwind CSS y los lenguajes de diseño utilizados en la aplicación. Sigue estas directrices estrictamente para crear la landing page:

1. **Estética General y Fondo**
   - El diseño debe ser futurista, oscuro y totalmente responsivo.
   - Utiliza un fondo de cielo espacial nocturno lleno de estrellas brillantes.
   - Integra un gran arco de luz degradada en tonos púrpuras neón y azules profundos que enmarque la sección principal (Hero).

2. **Cabecera Superior (Header / Navbar)**
   - **Izquierda**: Un logotipo tipográfico acompañado de un icono de red.
   - **Centro**: Menú de navegación principal.
   - **Derecha**: Un botón de acción (CTA) fuertemente destacado.

3. **Área Principal (Hero Section)**
   - **Título**: Grande, llamativo y utilizando un texto en degradado púrpura que domine la primera impresión de la pantalla.
   - **Subtítulo**: Un párrafo de texto descriptivo debajo del título principal.
   - **Botones de Acción**: Agrega dos botones debajo del párrafo. El principal debe estar relleno en púrpura neón, y el secundario debe tener solo el contorno (outline).

4. **Sección Inferior (Partners / Social Proof)**
   - Muestra una fila alineada horizontalmente de logotipos de empresas asociadas.
   - Los logotipos deben presentarse en color blanco (o tonos monocromáticos claros) para resaltar limpiamente sobre el fondo oscuro.

5. **Principios de Diseño**
   - Asegura una estética única y pulida que no parezca generada por IA.
   - Mantén todos los elementos limpios, profesionales y con espaciados (paddings/margins) propios del diseño vanguardista moderno.
